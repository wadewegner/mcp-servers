About This Demo

This is a demonstration of using the Model Context Protocol (MCP) to deploy static websites to DigitalOcean App Platform. The deployment was handled entirely through API calls, without using the DigitalOcean web UI or CLI tools.

The MCP server provides a set of tools that can be used by AI assistants to help you deploy and manage your static websites on DigitalOcean.
